# Calculator 🧮

A simple calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an easy-to-use interface.

## ✨ Features

- Addition, subtraction, multiplication, and division
- Clear (AC) button to reset the calculator
- Responsive layout for different screen sizes
- Clean and simple user interface
- Instant calculation of results

## 🛠️ Technologies

- HTML
- CSS
